Esta pasta contém as bases de dados utilizadas no projeto.



Os dados foram gerados a partir da coleta de notícias sobre o Palmeiras por meio de Web Scraping, além de etapas de tratamento, análise exploratória e processamento de texto.



Arquivos:



* noticias.csv: base consolidada com as notícias coletadas, contendo informações como título, texto, data e número de visitas.
* bow.csv: base com a representação textual das notícias utilizando Bag of Words (BoW).
* tfidf.csv: base com a representação textual das notícias utilizando TF-IDF.



Esses arquivos serão utilizados no projeto para o desenvolvimento dos modelos.
